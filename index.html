Android Application

News feed for Hannibal LaGrange Trojan Sports.

This is a screenshot from an Android application I created for my internship project. It is a basic news feed from the Hannibal LaGrange Trojan's athletic website. PC Hardware

Mod Project

This project was built for a PC Hardware class. The assignment was to build a mod project. As the name suggests, we were to take an item and modify it to serve a new purpose. We were also allowed to build something if we so desired. I decided to build a circuit that would blink to the music.

This project uses wire for the cirucuit, a couple of LEDs, a TIP31 component, a two position switch, and a P2 plug (to go in a headphone jack). The original project asks for 4 LEDs, however I only had 2 so I also added a resistor to the circuit to keep them from blowing. The instructions I followed for this project can be found at www.danielandrade.net
